Tutoriais de IA para Professores

Passo a passo simples para usar IA na sala de aula em 2026

Tutorial 1: Como usar Google Gemini

  1. Acesse gemini.google.com
  2. Faça login com conta Google (escolar funciona)
  3. Pergunte: "Crie plano de aula sobre frações para 6º ano"
  4. Copie e adapte o resultado

Tutorial 2: Corrigir redação com Copilot

  1. Acesse copilot.microsoft.com
  2. Cole o texto do aluno
  3. Pergunte: "Corrija esta redação ENEM e dê nota com justificativa"
  4. Use o feedback para conversa com o aluno

Tutorial 3: Gerar atividades inclusivas

  1. No Claude ou Gemini
  2. Pergunte: "Adapte esta atividade de matemática para aluno com dislexia"
  3. Receba versão com imagens, texto grande e áudio
